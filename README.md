### Running:

<aside>
⚠️ To work correctly, it is necessary to have the supabase tokens and add the .env file in the project root.
</aside>

To run the application in the development environment, just type:

```bash
# Step 1
npm install

# Step 2
npm run dev
```

### Stack:

- Typescript
- React

### Libs:

- Remix
